Tuning Tips

  • Always tune up to a note — come from below so the string tension is consistent
  • Play each string individually and let it ring clearly near the microphone
  • Tune in a quiet room for best accuracy — background noise affects detection
  • If a string is way off, tune to roughly the right pitch first then fine-tune
